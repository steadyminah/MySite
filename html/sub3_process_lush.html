LUSH

영국 프레쉬 핸드메이드 코스매틱 브랜드 ‘LUSH’를 리뉴얼 했습니다. 브랜드의 고유 컬러인 블랙&화이트는 가져가되 알록달록하고 자유로운 분위기가 잘 드러날 수 있도록 러쉬 프로덕트들을 위주로 디자인했습니다.

  • FORM
    반응형 웹사이트(PC, TABLET, PHONE)
  • DISPLAY GRID
    FLEX 사용

OVERVIEW

친환경적이면서도 자유분방한 분위기를 가진 러쉬의 특유 분위기를 살리고 싶어 정형화 되지 않은 랜딩페이지 레이아웃으로 디자인했습니다. 러쉬 오피셜 영상을 배경으로 사용하고, 로고플레이를 가운데 크게 배치함으로써 시원하면서도 브랜드를 각인시킬 수 있도록 제작했습니다. GNB는 페이지 상단에서는 투명했다가 스크롤이 내려오면서 고정되어 사용자가 언제든지 편리하게 메뉴 이동이 가능하도록 만들었습니다.

PROCESS PARTICIPATION

PLANNING
100%

리뉴얼 할 웹사이트를 선정하고, 기존에 가지고 있던 장점은 살리되 단점은 보완하는 방향으로 기획합니다.

DESIGN
100%

기업 아이덴티티를 고안하여 시각적 요소들을 하나의 룩앤필로 통일해 웹사이트의 디자인을 리뉴얼하였습니다.

PROGRAMMING
100%

HTML과 CSS를 기반으로 웹사이트를 설계했으며 CSS 및 JS를 활용한 동적효과를 더하여 퍼블리싱을 완성하였습니다.

MAINTENANCE
100%

전반적인 레이아웃과 화면구성을 검토하며 디테일한 부분들을 수정하고 보완하여 완성도 있는 마무리를 합니다.

COLOR SYSTEM

브랜드의 고유 컬러 블랙과 화이트를 베이스로 하고 다른 요소들은 러쉬의 제품으로 다채로움을 더했습니다.

TYPOGRAPHY

서체는 고객들에게 기업의 이야기를 전달하는 역할을 합니다. 사용한 서체는 가독성이 좋은 Noto Sans와 러쉬의 분위기를 잘 담아내는 필기체인 parkway lush를 사용했습니다. 타이포그래피의 올바른 활용은 기업의 이미지를 보다 더 친절하고, 신뢰성 있는 브랜드로 만들어줍니다.
